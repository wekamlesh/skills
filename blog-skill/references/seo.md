# SEO and Visuals Reference

## SEO Requirements

- Write in a programmatic SEO style when useful — without sounding spammy
- Use strong titles, relevant headings, semantic keywords, and structured sections
- Create natural keyword coverage — avoid forced keyword stuffing
- Include meta title and meta description when requested
- Generate FAQ sections when useful for search intent
- Make structure easy to scan and index

## Visuals and Charts

- Identify where charts, tables, graphs, or visual comparisons will improve the article
- Create chart-ready data summaries from researched metrics
- Suggest the best chart type for each insight (bar, line, table, comparison grid, etc.)
- Default visual style: clean pastel or dark gray + neon depending on article tone
- Visuals should feel minimalist, modern, and premium
- If the model cannot generate actual images: provide precise chart specs, labels, and data tables ready to hand off to a designer or chart tool
- Keep charts aligned with the article's story — not decorative only

## Title Guidance

Strong titles should:
- Be specific and promise clear value
- Use numbers, comparisons, or strong verbs when natural
- Target the primary keyword near the front
- Avoid clickbait — stay honest to what the article delivers

## Meta Description

- 150–160 characters
- Summarize the article's core value
- Include the primary keyword naturally
- Write in active voice
